h3: "Tento ukázkový kód ukazuje konverzi EPS na PS Java"

item2: Načtěte soubor EPS v instanci PsDocument prostřednictvím streamu
item3: Vytvořte a nastavte instance tříd ImageSaveOptions a ImageDevice
